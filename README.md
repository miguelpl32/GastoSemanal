# GastoSemanal
Calcular Gasto Semanal

## Objetivo
+ Añadir presupuesto Inicial
+ Restas cada compra del presupuesto
+ Al eliminar una compra se vuelve añadir al presupuesto
+ Si se excede el presupuesto no permite comprar
